# Zoom 취약점 분석 - 데이터 유출 방지

## 취약점 개요
- 회의 메타데이터 (참여자 정보, 시간) 서버 저장
- 클라우드 백업 데이터 노출 위험
- URL 유출로 인한 무단 참여 가능

## 주요 취약점
1. **메타데이터 노출**: 참여자 목록, 회의 시간 저장
2. **클라우드 데이터**: 제3자 접근 가능
3. **URL 유출**: 회의 링크 탈취 위험

## 보완方案 (Jitsi Meet 기반)
- 메타데이터 최소화
- 데이터 익명화 처리
- 임시 URL 생성

## 구현 파일
- `metadata_protection.py`: 메타데이터 보호
- `data_masking.py`: 데이터 마스킹
