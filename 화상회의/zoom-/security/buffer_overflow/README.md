# Zoom 취약점 분석 - 버퍼 오버플로우

## 취약점 개요
- 사용자 입력 검증 부재
- 파일 전송 시 취약점 존재
- 이전 버전에서 버퍼 오버플로우 발견

## 주요 취약점
1. **입력 검증 미흡**: 길이 제한 없음
2. **파일 취약점**: 악성 파일 업로드 가능
3. **메모리 손상**: 버퍼 오버플로우 탐지

## 보완方案 (Jitsi Meet 기반)
- 입력 길이 제한
- 파일 검증 (MIME 타입 검사)
- 샌드박스 환경 실행

## 구현 파일
- `input_validation.py`: 입력 검증
- `sandbox.py`: 샌드박스 실행
